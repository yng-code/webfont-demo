混植では Latin と日本語の字面サイズが違い、英数字がやや小さく見えることがあります。Adjust を ON にすると Alphabet の見かけが 日本語 に近づきます。

追加パターン：Gen Interface JP（単一書体で混植済み）

Inter と Noto Sans JP を1つの書体に統合し、和欧の見え方を最初から揃えた UI 向けフォント。font-family に1つ指定するだけで済みます。
